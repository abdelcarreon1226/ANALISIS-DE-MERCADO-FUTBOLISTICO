# ANALISIS-DE-MERCADO-FUTBOLISTICO
Este proyecto consiste en analizar información de jugadores y equipos de fútbol con el objetivo de identificar patrones relacionados con el valor de mercado, edad, tamaño de equipos y diferencias entre ligas.

# OBJETIVO DEL PROYECTO
El objetivo principal es analizar cómo diferentes características de jugadores y competiciones pueden influir en el valor de mercado dentro del fútbol.

# PREGUNTAS DEL NEGOCIO
- ¿Qué liga tiene los jugadores con mayor valor promedio?
- ¿Cuál es la edad promedio de los jugadores?
- ¿Qué equipos concentran mayor valor de mercado?
- ¿Existe relación entre edad y valor económico?
- ¿Qué competiciones tienen jugadores más jóvenes?

## DATASET 
El contenido de esta dataset fue recopilado de kaggle contiene informacion de valor de mercado de equipos de diferentes ligas, edad, promedio de cada equipo, y profundidad de plantilla.
